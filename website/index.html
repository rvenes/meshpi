Det viktigaste

Bygd for å stå på heile tida

Daemonen eig sambandet. Terminalgrensesnittet kan opnast og lukkast utan at meldingar går tapt.

01

Sanntidschat

Public kanal 0 og direkte meldingar dukkar opp automatisk, med RF/MQTT, SNR, RSSI og ACK når data finst.

02

Heile nodelista

Naviger med Tab og piltastar, sjå detaljar og trykk Enter for å opne DM med ein node.

03

TCP og USB

Byt mellom lagra eller oppdaga Meshtastic-nodar utan å starte bakgrunnstenesta på nytt.

04

Lokal historikk

Meldingar, ulest-status og nodeinformasjon blir lagra lokalt i SQLite. Arkivering slettar ingenting.

Kom i gang

Vel kor lenge bakgrunnen skal køyre

Standard er always: meldingar kjem inn også når terminalgrensesnittet er lukka. Vel session dersom daemonen berre skal køyre medan du bruker MeshPi.

Linux og Raspberry Pi OS

Installerer under /opt/meshpi og set opp ei avgrensa systemd-teneste (always).

curl -fLO https://venes.org/meshpi/install-linux.sh && less install-linux.sh && sudo sh install-linux.sh

Session (utan sudo): sh install-linux.sh --mode=session

macOS

Installerer for gjeldande brukar og held daemonen i gang med launchd (always).

curl -fLO https://venes.org/meshpi/install-macos.sh && less install-macos.sh && sh install-macos.sh

Session: sh install-macos.sh --mode=session

Windows PowerShell

Installerer per brukar og bruker ein prosessvakt som startar daemonen ved innlogging og etter krasj.

Invoke-WebRequest https://venes.org/meshpi/install-windows.ps1 -OutFile install-windows.ps1; Get-Content .\install-windows.ps1; .\install-windows.ps1

Etter installasjon

  1. Opne ein ny terminal.
  2. Køyr meshpi. Ved første oppstart opnar nodeveljaren automatisk.
  3. Vel ei oppdaga TCP-/USB-eining, eller skriv målet manuelt.

Installeren lagar ei ny versjonsmappe, kontrollerer pakken og alle avhengigheiter med SHA-256, køyrer ein offline sjølvtest og byter først deretter. Ved mislykka helsesjekk blir førre versjon sett tilbake automatisk.

Les skriptet før du køyrer det

Dei korte kommandoane over stoler på det serveren leverer akkurat då. Du kan i staden laste ned skriptet, lese det og køyre den lokale fila:

curl -fLO https://venes.org/meshpi/install-linux.sh
less install-linux.sh
sudo sh install-linux.sh

curl -fLO https://venes.org/meshpi/install-macos.sh
less install-macos.sh
sh install-macos.sh

Invoke-WebRequest https://venes.org/meshpi/install-windows.ps1 -OutFile install-windows.ps1
Get-Content .\install-windows.ps1
Set-ExecutionPolicy -Scope Process Bypass; .\install-windows.ps1

Windows session-modus: .\install-windows.ps1 -Mode Session.

Oppdatering

Same kommando. Ingen datatap.

Køyr installasjonskommandoen på nytt. Eksisterande konfigurasjon, tilkoplingsprofilar og SQLite-historikk blir tekne vare på.

MeshPi sjekkar version.json ved oppstart. Når ein ny versjon finst, blir rett kommando vist som ei lokal systemmelding i chatloggen. Han blir aldri sendt til mesh-nettet.

Kontrollert nedlasting

  • Manifest og installasjon over HTTPS
  • SHA-256 for MeshPi-pakken og låste avhengigheiter
  • Offline sjølvtest og automatisk rollback
  • Ingen automatisk køyring av oppdateringar
  • Ingen telemetri eller konto

Manifestet er digitalt signert med ein separat RSA-3072-nøkkel som er festa i appen og alle installatørane. Endra eller usignerte manifest blir avviste før pakkar blir installerte.

Bakgrunn og diagnostikk

Styr daemonen utan å leite i systemmenyar

meshpi service status
meshpi service start
meshpi service stop
meshpi doctor --offline

I session-modus startar MeshPi daemonen ved behov. Når du avsluttar TUI-en, kan du velje om han skal stoppast eller stå att.

Avinstaller utan datatap

Standard avinstallering fjernar program og autostart, men bevarer konfigurasjon, historikk og profilar.

  • Linux / Pi – bruk --purge-data for full sletting
  • macOS – bruk --purge-data for full sletting
  • Windows – bruk -PurgeData for full sletting

Detaljerte mapper, loggkommandoar og drift står i README-en.
