Det viktigaste

Bygd for å stå på heile tida

Daemonen eig sambandet. Terminalgrensesnittet kan opnast og lukkast utan at meldingar går tapt.

01

Sanntidschat

Public kanal 0 og direkte meldingar dukkar opp automatisk, med RF/MQTT, SNR, RSSI og ACK når data finst.

02

Heile nodelista

Naviger med Tab og piltastar, sjå detaljar og trykk Enter for å opne DM med ein node.

03

TCP og USB

Byt mellom lagra eller oppdaga Meshtastic-nodar utan å starte bakgrunnstenesta på nytt.

04

Lokal historikk

Meldingar, ulest-status og nodeinformasjon blir lagra lokalt i SQLite. Arkivering slettar ingenting.

Kom i gang

Éin kommando

Vel operativsystem. Installeren kontrollerer SHA-256 før pakken blir installert.

Linux og Raspberry Pi OS

Installerer under /opt/meshpi og set opp ei avgrensa systemd-teneste.

curl -fsSL https://venes.org/meshpi/install-linux.sh | sudo bash

macOS

Installerer for gjeldande brukar og held daemonen i gang med launchd.

curl -fsSL https://venes.org/meshpi/install-macos.sh | bash

Windows PowerShell

Installerer for gjeldande brukar og startar daemonen skjult ved innlogging.

powershell -NoProfile -ExecutionPolicy Bypass -Command "irm https://venes.org/meshpi/install-windows.ps1 | iex"

Etter installasjon

  1. Opne ein ny terminal.
  2. Køyr meshpi new og vel TCP-node eller USB-port.
  3. Køyr meshpi for å opne terminalgrensesnittet.
Oppdatering

Same kommando. Ingen datatap.

Køyr installasjonskommandoen på nytt. Eksisterande konfigurasjon, tilkoplingsprofilar og SQLite-historikk blir tekne vare på.

MeshPi sjekkar version.json ved oppstart. Når ein ny versjon finst, blir rett kommando vist som ei lokal systemmelding i chatloggen. Han blir aldri sendt til mesh-nettet.

Trygg nedlasting

  • Manifest og installasjon over HTTPS
  • SHA-256 blir kontrollert før installasjon
  • Ingen automatisk køyring av oppdateringar
  • Ingen telemetri eller konto
